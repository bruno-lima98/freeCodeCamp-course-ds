# freeCodeCamp - Data Science Course

Study repository with exercises and mini-projects from the freeCodeCamp Data Science couse, focused on **pandas** fundamentals for data manipulation and analysis.

## 🎯​ Goal

Build a solid pandas foundation (reading, filtering, extraction and visualization of data) before moving into applied Machine Learning and fraud detection projects.

## 📁 Structure

| Folder                                    | Content                                                                |
|-------------------------------------------|------------------------------------------------------------------------|
| `01-intro-to-pandas`                      | Introduction to pandas: Series, DataFrames, basic operations           |
| `02-project-web-scraping`                 | Mini-project: web scraping for data collection                         |
| `03-filtering-data`                       | Data filtering and selection with pandas                               |
| `04-data-extraction`                      | Data extraction from different sources                                 |
| `05-pivot-table`                          | Pivot tables and aggregations                                          |
| `06-project-data-visualization`           | Mini-project: data visualization and interactive visualization         |
| `07-groupby-and-aggregate-function`       | GroupBy and aggregation functions in dataframes                        |
| `08-merging-and-concatenating-dataframes` | Merging and concatenating multiples dataframes                         |
| `09-regular-expressions`                  | Pattern matching and text extraction using regular expressions         |
| `10-project-data-cleaning`                | Mini-project: complete data cleaning process                           |
| `11-machine-learning`                     | Introduction of machine learning concepts                              |
| `12-project-text-classification`          | Mini-project: introduction to a practical machine learning application |

## 🛠️ Tech Stack

- Python
- Pandas
- Machine Learning (Introductory concepts)
- Jupyter Notebook
